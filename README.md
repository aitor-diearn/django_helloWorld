# django_helloWorld

### Español

Este proyecto muestra un mensaje de 'Hello World' en la página principal.

Este proyecto se ha hecho usando el framework de Django, versión 6.0.2, y un entorno virtual, creado con venv.

___

# English

This project shows you a 'Hello World' message in the principal page.

This project is done with Django framework, version 6.0.2, and a virtual environment, created with venv.
